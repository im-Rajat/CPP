# Miscellaneous 

- We can use exit(0); for termination of a program.

### Different Indentation Style

- Allman / BSD
    ```cpp
    while (x == y)
    {
        something();
        something_else();
    }
    ```
- K&R
    ```cpp
    while (x == y) {
        something();
        something_else();
    }
    ```
- There are many like: GNU, Whitesmiths, Ratliff, Horstmann, Pico, Lisp, APL

- Reference : https://en.wikipedia.org/wiki/Indentation_style
